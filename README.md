# Catch The Box Game

A fun browser-based game where players try to catch a randomly moving box before the timer runs out.

## Features
- Score tracking
- Timer system
- Responsive design
- Interactive gameplay

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Open the game in your browser.
2. Click on the moving box to score points.
3. Try to get the highest score before time runs out.
